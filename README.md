# Azure Networking Lab (AZ-104)

Hands-on lab project covering the Networking domain of the AZ-104: Microsoft Azure Administrator certification. Built in a live Azure environment (Student subscription, Switzerland North) with full documentation of design decisions, configurations, and troubleshooting.

## 🎯 Purpose
Practical, portfolio-ready demonstration of Azure networking skills: VNet design, peering, security, routing, load balancing, DNS, VPN connectivity, and diagnostics — as preparation for both the AZ-104 exam and real-world cloud administration.

## 🗂️ Structure
Each step documents one networking concept: design decisions, Portal + CLI implementation, verification, and key learnings.

| Step | Topic |
|---|---|
| 1 | VNet & Subnet Design |
| 2 | VNet Peering |
| 3 | Network Security Groups & ASGs |
| 4 | Route Tables & UDRs |
| 5 | Azure DNS |
| 6 | Load Balancer |
| 7 | Application Gateway |
| 8 | VPN Gateway |
| 9 | Network Watcher & Diagnostics |
| 10 | Azure Bastion & NAT Gateway |
| 11 | Final Review & Architecture Summary |

## 🛠️ Environment
- Azure Student Subscription (region-locked: `switzerlandnorth`)
- Naming convention: CAF-aligned, `chn` regional suffix
- Tools: Azure Portal, Azure CLI (VS Code)
